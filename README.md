# Extension
# Name:
BMI Calulator
# Descripation: 
Calulates the  BMI  using a person's height and weight.
# Installation Guide
1.Open the Chrome Web Store.<br>
2.Find and select the extension.<br>
3.On the developer mode.<br>
4.Then  click unpacked and select the folder.
